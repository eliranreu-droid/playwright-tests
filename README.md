# Playwright Automation Project - SauceDemo

This project is an automated testing suite for the [SauceDemo](https://www.saucedemo.com/) website. It uses **Playwright** with **TypeScript** and follows the **Page Object Model (POM)** architecture.

## 🚀 Features
* **Full Purchase Flow:** Automated test for logging in, adding dynamic products to the cart, and completing the checkout.
* **Page Object Model:** Separated locators and logic for better maintenance.
* **Dynamic Cart Validation:** Verifies cart badge counts based on added items.

## 🛠️ Tech Stack
* **Framework:** Playwright
* **Language:** TypeScript
* **Pattern:** Page Object Model (POM)
